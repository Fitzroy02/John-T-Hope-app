# README: Mr Walker Licensing Framework

## Overview
Mr Walker is a fictional character created and owned by **Fitzroy B. Edwards**, writing as **John T. Hope**.
Hopenism uses the character under a **dual-stream licence** designed to protect the IP while enabling symbolic educational use.

---

## 1. Appearance Licence Fee
Hopenism pays a discounted annual fee for the right to include Mr Walker in the app.
This covers:

- visual appearance
- textual references
- symbolic use
- educational modules

---

## 2. Performance-Based Royalty
Mr Walker earns a royalty for each user interaction involving the character.
This mirrors the artist streaming rate used within the Hopenism platform.

Interaction types include:

- completed modules
- animations
- guided exercises
- narrative sequences

---

## 3. Activation
The licence activates only when Mr Walker appears in the app.
Before that, no fees or royalties apply.

---

## 4. Ownership
Mr Walker remains the exclusive IP of **Fitzroy B. Edwards (John T. Hope)**.
Hopenism receives usage rights only.

---

## 5. Purpose
This structure ensures:

- legal clarity
- brand separation
- IP protection
- scalable future media development
