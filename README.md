## How to Run Locally

### Requirements
- Python 3.8+
- Streamlit

### Installation

1. Clone the repository
   ```bash
   git clone https://github.com/Fitzroy02/John-T-Hope-app.git
   cd John-T-Hope-app
   ```

2. Install dependencies
   ```bash
   pip install -r requirements.txt
   ```

3. Run the application
   ```bash
   streamlit run streamlit_app.py
   ```
